Description

Enum pour les types d'indexes dans le MT : E : index dont la valeur est estimee (lors de l'apparition de trous de consommation) R : index dont la valeur est reelle (envoyee au MT) S : index dont la valeur est simulee (pour le jour J si aucune valeur n'est recue) Created by a454063 on 27/10/2015.
