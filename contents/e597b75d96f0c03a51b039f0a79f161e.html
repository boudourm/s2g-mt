getConsoDaily

UMLOperation
S2G-MT :: S2G-MT-Services :: net :: atos :: s2g :: mt :: services :: producer :: ConsoService :: getConsoDaily

Description

Récupération de parametre en base Ce parametre permet de borner la date fin des données (il faut que les donnees soient validées par le systeme) L'occupant doit occuper un logement de type private L'occupant est bien présent dans son appartement pour le mois et l'année demandée Récupération des consommations. Recupération des données PRIVEES du mois précedant Recupération des données COLLECTIVES du mois précedant Récupération des consommations pour tous les fluides pour une période donnée @param premise le logement à considérer @param startDate date de début des consos à récupérer @param endDate date de fin de consos à récupérer @param occupyStartDate Date de début d'occupation du logement par l'utilisateur @param occupyEndDate Date de fin d'occupation du logement par l'utilisateur @param prevDateBegin la date de début de la période précédente, pour calculer les tendances @param prevDateEnd la date de fin de la période précédente, pour calculer les tendances @return
