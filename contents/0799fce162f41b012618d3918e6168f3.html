getLastExistingPCSBeforeTheGivenDate

UMLOperation
S2G-MT :: MT-datamodel :: net :: atos :: ecocite :: datamodel :: dao :: GasPowerRepository :: getLastExistingPCSBeforeTheGivenDate

Description

Retourne le dernier PCS existant avant la date donnee en param. Peux retourner la veille, ou la valeur la plus recente avant la date donnee. Null sinon. @param p_Date la date reference, le PCS du jour retourne devra etre le PCS d'un jour anterieur a la date reference @return le dernier PCS enregiste anterieure a la date donnee en parametre
